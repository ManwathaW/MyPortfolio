## ?? Troubleshooting

### Problem: Sitemap shows 404
```
Solution:
1. Verify sitemap.xml in wwwroot/ folder
2. Redeploy to Railway
3. Check: https://manwathawanga.up.railway.app/sitemap.xml
4. Should see XML content in browser
```

### Problem: Google won't verify ownership
```
Solution:
1. HTML file method:
   - Download verification file
   - Save to wwwroot/
   - Run: git add . && git commit -m "Add Google verification"
   - Push: git push origin master
   - Wait 5 mins for deployment
   - Try verifying again
2. If still fails:
   - Try Google Tag Manager method
   - Or domain verification (if using custom domain)
```

### Problem: Crawl errors in Search Console
```
Solution:
1. Check specific error in Search Console
2. Look at URL path
3. Verify page exists
4. Check for redirects
5. Resubmit in URL Inspection
```

### Problem: Still not in Google after 4 weeks
```
Solution:
1. Check Search Console for issues
2. Verify Core Web Vitals
3. Build backlinks (share on social media)
4. Add more content (blog posts)
5. Be patient - can take 6 months for brand new sites
```
